## 1. 계산 공식 (4단계)

```
① 항목 유사도 sim_k        : 유형별 계산 (연속/범주/이진/순서형)
② 항목 conf_k             : 항목별 기본 conf × 데이터 신뢰도계수(낮은쪽)
③ 카테고리 점수            : Σ(sim_k × conf_k) / Σ(conf_k)
④ 종합 점수               : Σ(카테고리점수 × 카테고리가중치) / Σ(가중치)
```

카테고리 가중치: 시장 0.25 / 규제 0.25 / 금융 0.20 / 시스템 0.30

```
유효신뢰도 = Σ(카테고리가중치 × 평균conf) / Σ(가중치) = 0.68

```

## 6. 해석

- **시장(62.8)이 가장 높음**: 판매규모·세그먼트·구매주기 등 구조 지표가 유사.
- **시스템(41.3)이 발목**: 영국 Alfa/SaaS/PCP vs 폴란드 Comarch/SI/리스 — 코어 이식 부적합. 게이트가 이를 차단.
- **유사도 0점 항목들**이 점수를 끌어내림: 라이선스 체계, 법인형태, 신용정보 인프라, 추심·회수 절차, 결제 인프라 — 영국과 폴란드의 제도·인프라가 근본적으로 다른 지점들.
- **결론**: 시장은 비슷해도 시스템·제도 인프라가 달라 "영국 시스템 이식형" 진출은 부적합. 폴란드는 법인 리스 중심 진출국을 비교 기준으로 삼는 것이 타당.
