## 공통사항
- 전체 화면의 상단바만 존재하고 왼쪽 오른쪽 사이드바는 없어야함.
- 전부 한글로 구성, 단 화면 최상단 우측에 영문 변환 가능한 버튼 구성
- 영문 변환 시 전체 한글->영문으로 변환 필요

## 메인화면 (M1)
### 디자인
- 세계지도가 메인 가장 큰 영역으로 표시되어야함.
- 세계지도는 지적도 형식이 아닌 플랫 형태의 지도 필요
- 진출국가, 진출예정국가 마크로 지도에 표시(국가의 수도 위치에 마크 필요)
- 지도 최우측 상단에 범례표시로 진출국가, 진출예정국가 분류 필요 (적절한 투명도 조절 필요, 크기는 작게 표시)
- 권역 단위도 선택 가능하도록 구분 필요 (유럽, 남미, 북미, APEC 등)
- 권역 선택 시 권역 부분만 선택 강조 필요 + 해당 권역만 확대되는 시뮬레이션 필요 + 해당 권역 우측 상단에 x 닫기 버튼 표시
  - 닫기 클릭 시 전체 세계지도 화면으로 축소
- 상단 메뉴바 구성(각 메뉴별 드롭다운)
  - 메인
  - 상세
    - 권역 상세(M2)
    - 국가 상세(M3)
    - 국가 전체 보기(M7)
  - 보고서
    - 권역 보고서(M4)
    - 국가 보고서(M5)
  - 설정
    - 가중치 설정(M6)
  - 국기 두개 나란히
    - 태극기, 영국국기
- 최상단우측 영역에 카드 형태의 프로그레스바 패널 노출/비노출 기능 필요
  - 국가명 또는 권역명 표시
  - 분석 진행중 표시
  - 진행률 표시
  - 패널 하단에 상세보기 버튼 구성
    - 프로그래스 팝업(P3) 이동
  - 패널 하단에 보고서 버튼 구성
    - 진행률 100%의 경우 활성화
    - 국가 단위, 권역 단위 선택 기준에 따라 M4 또는 M5 화면으로 이동

### 기능
- 지도 내 권역 선택 시 
  - 해당 권역 지도 확대 애니메이션 추가
  - 카드 형태의 팝업(P1) 노출
- 지도 내 국가 선택 시
  - 카드 형태의 팝업(P2) 노출
- 우측 국기 선택 시,
  - 태극기: 언어 설정 '한국어'로 변경
  - 영국국기: 언어 설정 '영어'로 변경

## 설정 화면 (M6)
### 디자인
- 화면 우측 상단에 가중치 룰셋 ID 드롭다운
- 화면 우측 하단에 [저장] 버튼
- 총 3개의 패널로 구성
  - 카테고리 가중치
    - 총 합계 100% 로 표현되는 슬라이더(왼쪽 0, 오른쪽 100%)
      - 시장
      - 규제
      - 환경(금융)
      - 시스템
  - 임계값 신뢰도 계수: 각 계수는 0부터 100까지
    - 이식 임계
    - 시스템 게이트
  - 출처 신뢰도 계수: 0부터 1.0 까지
    - Tier 1(공식)
    - Tier 2(준공식)
    - Tier 3(참고)

### 기능
- 룰셋 ID 드롭다운 클릭 시
  - 저장된 룰셋 ID 목록 출력
    - 룰셋 ID 선택시 ~~ 데이터 가져와 화면에 설정
- 카테고리 가중치
  - 하위의 슬라이더 값은 총 100% 여야 함
  - 각 항목의 수치 변경 시 ~~ 파일 json 값 변경

## 프로그레스 팝업 (P3)
### 디자인
- 총 4개의 프로그레스 바가 병렬 구성
  - 시장
  - 규제
  - 상품
  - 시스템
  - 결과 생성
- 히든 버튼으로 우측 하단에 [보고서보기] 버튼
- 우측 상단에 [X] 닫기 버튼

### 기능
- 각 에이전트별 프로그레스바에 진행률 표시
  - 개별 유사성 검증: 유사성 검증 스크립트(SC)를 실행하는 에이전트 4개 호출
    - 시장
    - 규제
    - 상품
    - 시스템
  - 개별 유사성 검증이 끝나면, 결과 생성 에이전트 실행
- [보고서보기] 버튼 클릭 시, 진단결과 화면(M5) 화면으로 이동
